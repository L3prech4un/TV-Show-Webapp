Introduction¶

What is TV-Show-Webapp?¶

TV-Show-Webapp is a Flask-based web application that allows users to:

  • Create profiles and share their TV/movie watching habits

  • Post reviews and comments on media

  • Follow other users and see their activity feeds

  • Track shows they’re watching, have watched, or want to watch

  • Discover new shows through community recommendations

Features¶

  • User Profiles: Create accounts with custom usernames and profiles

  • Social Feed: See posts from users you follow

  • Media Tracking: Organize shows into “Watching”, “Watched”, and “Watchlist” categories

  • Posting System: Share reviews and thoughts about shows

  • Follow System: Connect with other TV/movie enthusiasts

  • Search Functionality: Find users and content

Technology Stack¶

  • Backend: Python, Flask, SQLAlchemy

  • Database: PostgreSQL

  • Frontend: HTML, CSS, JavaScript

  • Deployment: Docker, Docker Compose

  • Documentation: Sphinx (what you’re reading now!)
